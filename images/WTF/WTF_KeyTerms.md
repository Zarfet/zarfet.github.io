# WHAT THE FUTURE! - KEY GAME TERMS

---

**Scenario:** The future crisis you must solve this round.

**Tool:** Your method/approach (e.g., Blockchain, Low-Tech, AI).

**Constraint:** The limitation you must respect (e.g., No Electricity, Total Privacy).

**Challenge:** The specific problem to solve in the Scenario.

---

**Design Phase (5 min):** Create your solution using 1 Tool while respecting 1 Constraint.

**Presentation (30 sec):** Explain your solution to everyone.

**Q&A (1 min):** Answer questions from the other team.

**Upload (1 min):** Submit solution to the board for AI evaluation.

---

**Validation:** 3 AIs evaluate your solution (1-5 each on 5 criteria). Consensus = your points.

**Evaluation Criteria:**
1. Solves the challenge
2. Meets the constraints
3. Uses the tool
4. Low-risk
5. Easy to execute

**Max score:** 25 points per round

---

**Power Card:** Magic card that changes the game (e.g., Double Down, Point Swap).

**Losing Team:** Team with fewer total points after a round. Draws 1 Power Card. Max hand: 2 cards. Decides to use it or save it.

**Round:** One complete cycle: Scenario + Design + Validation. Game has 4 rounds.

---

**Winner:** Team with most total points after Round 4.
